Reaction Checker
This is a simple reaction checker.

This program measures your reaction speed based on how fast you click the screen. It uses java script, css and html coding. I made the main functionaity using java and made the website design using css. This program also shows your average reaction time by calculating the averag of 3 scores.

How to play: 
To start the game click anywhere on the screen.
A red window appears. Wait till it turns green and click as soon as it turns green.
The system will measure your speed and display it. To play again click anywhere on the screen. After 3 tries it will show the average score.
If you click while the screen is red it will say too soon. To try again click anywhere on the screen.
